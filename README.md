## Scottish Mountains Explorer

Interactive browser for Scotland's hills — Munros, Corbetts, Donalds & Grahams — with ratings, parking info, and trip planning from Edinburgh.

- [Mountains Explorer](https://scotland-mountains.github.io/mountains/mountains.html) — sortable, searchable table of 808 mountains
- [Mountains Map](https://scotland-mountains.github.io/mountains/map.html) — google map of 808 mountains
- [Hiking Guide](https://scotland-mountains.github.io/mountains/guide.html) — preparation guide with weather, kit, safety & emergency numbers

Data sourced from [WalkHighlands](https://www.walkhighlands.co.uk/).
